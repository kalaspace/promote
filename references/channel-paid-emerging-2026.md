# Paid Emerging Channels 2026 — CTV/OTT, TikTok Shop, Retail Media, DOOH

> **Channel scope** : couvre **CTV/OTT** (Hulu, Roku, Netflix Ads tier), **TikTok Shop Ads**, **Walmart Connect + Retail Media Networks** (Kroger, Target, Amazon hors classique), **Programmatic DOOH**. Ne couvre PAS Amazon Ads classique (Sponsored Products) qui a son operator dédié, ni TikTok Ads classique.
> **Why framework, not operator** : émergents fort 2024-26 mais pas d'individu-doctrine distillable encore. Doctrines en formation. WAIT v1.2 sur operators dédiés quand les figures émergent.
> **Used by** : `promote-channel-strategist` quand le brand est en scaling >$5M/an et le mix paid mature exige diversification au-delà de Meta/Google.

---

## Quand consulter ce framework

**Fit conditions** :
- Brand DTC ou CPG avec >$5M revenue annuel et budget paid >$500K/an.
- Mix Meta/Google déjà saturé (CACs en hausse, plateau ROAS).
- Brand-aware budget ≥20% du total media spend.
- Pour Walmart Connect / retail media : produit physique vendu sur retailer + Amazon.

**Anti-fits** :
- Pre-PMF startup : ces canaux exigent maturité product + supply chain.
- Pure-play SaaS B2B : aucun de ces canaux n'est fit (sauf CTV pour brand-led B2B at scale, niche).
- Budget <$50K/mois sur le canal : sub-scale, pas de signal exploitable.

## Tactiques 2026

### CTV / OTT (Hulu, Roku, Netflix Ads tier)

1. **Hulu Ad Manager + Disney DSP** : entry-level CTV, self-serve, minimum $500-2K/campaign. Bon pour test initial.
2. **Roku OneView** : audience-data-rich (Roku owns ACR data sur 80M+ households US). Meilleur pour reach + frequency control.
3. **Netflix Ads tier** (lancé fin 2022, scale 2024+) : premium inventory, CPMs $40-65 USD, minimum spend $2-5K. Top-of-funnel pour brand-led DTC scaling.
4. **Programmatic CTV via The Trade Desk / DV360** : meilleure flexibilité audience + mesure incremental, mais setup plus lourd.
5. **Genre/show targeting** : éviter le broad CTV. Cibler shows-fit avec ta cible (sport/news pour finance, drama/comedy pour DTC mass).
6. **Frequency cap discipliné** : 3-5 expositions/semaine max. Au-delà = ad fatigue + waste.

### TikTok Shop Ads

1. **Shop Ads in-feed** : commerce-driven, mécanique différente de TikTok Ads classique. Lien direct produit → checkout in-app.
2. **Live Shopping Ads** : amplification de TikTok Lives sponsorisées par le brand. Fit produits demo-friendly (beauty, gadgets, fashion).
3. **Affiliate / creator integration MANDATORY** : TikTok Shop sans creator partnership = ROAS catastrophique. Setup creator commission 10-20% + paid amplification.
4. **Video Shopping Ads** : creative <30s, hook <3s, CTA "Shop now" claire. Format vertical 9:16 obligatoire.
5. **Product seeding via Shop Affiliates** : envoyer du produit gratuit à 50-200 creators tier-2 (10K-100K followers) avant de scale paid.

### Walmart Connect + Retail Media Networks (Kroger, Target, Amazon DSP off-Amazon)

1. **Walmart Sponsored Search** : search ads sur Walmart.com (équivalent Amazon Sponsored Products). Setup obligatoire si tu vends sur Walmart. Le vrai growth area 2024-26 paid.
2. **Walmart Display + DSP** (off-site programmatic backed par Walmart 1P data) : reach external avec Walmart purchase data. Premium pricing mais incrementality élevée.
3. **Kroger Precision Marketing** (Kroger 84.51° data) : pour CPG food/beverage/grocery, accès à 60M+ household purchase data.
4. **Target Roundel** : equivalent Target. Fit DTC fashion/home/beauty + Target.com presence.
5. **Incrémentalité testing OBLIGATOIRE** : retail media + Amazon Ads = double-spend potential. Holdout tests par retailer avant scaling.
6. **Closed-loop attribution** : utiliser le retailer's own measurement (RoAS direct sur retailer.com), pas pixel cross-platform.

### Programmatic DOOH (Digital Out-of-Home)

1. **Status 2026 = WAIT v1.3** : marché fragmenté (Vistar Media, Place Exchange, Hivestack), operators émergent slow.
2. **Quand commencer à test** : >$1M media spend total, brand budget Q1 2027+, key urban markets fit (NYC, LA, SF, London, Paris).
3. **Use case fort** : geo-fenced DOOH (proche d'un retail store, événement, concurrent location) avec mobile retargeting downstream.
4. **Buy via DSPs déjà connus** : The Trade Desk, DV360, Yahoo DSP supportent DOOH inventory désormais. Pas besoin de plateforme dédiée pour test.
5. **Mesure faible 2026** : attribution DOOH reste lift-study based, pas click-based. Budget brand, pas performance.

## Cadence + format

**CTV/OTT** : campaigns flighted (4-8 semaines), 15s + 30s spots, vertical NON-pertinent (CTV = horizontal 16:9). Anti-formats : verticals coupés, voiceover-only sans visual narrative.

**TikTok Shop Ads** : daily campaigns continues, creative refresh tous les 7-14 jours (fatigue rapide), vertical 9:16 hook <3s. Anti-formats : horizontal, voiceover corporate, pas de creator face camera.

**Retail Media** : always-on Sponsored Search + bursts Display autour de retail moments (Walmart Days, Target Circle Week, Kroger seasonal). Anti-formats : creative Amazon-imported sans adaptation retailer.

**DOOH** : campaigns courts 1-4 semaines, creative high-contrast lisible <3s, geo-fenced. Anti-formats : creative chargée texte, faible contraste, verticals.

---

## Anti-patterns 2026

1. **Performance-marketing-only sur CTV** : c'est upper-funnel, pas direct response. Mesurer ROAS click-based = mauvais fit. Mesurer brand lift + incremental sales.
2. **TikTok Shop sans creator partnership** : le scale-driver de TikTok Shop EST le creator. Paid-only sans creator commission tier = ROAS catastrophique.
3. **Double-spend Amazon Ads + Walmart Connect sans incrémentalité testing** : risk de cannibalize. Holdout tests obligatoires sur 4-8 semaines.
4. **CTV au minimum spend** : <$2K/campaign = sub-scale, frequency caps non-tenables, pas de signal exploitable.
5. **DOOH sans mobile retargeting downstream** : DOOH seul = brand spray. DOOH + mobile retargeting geo-fenced = vrai funnel.
6. **Netflix Ads tier sur creative recyclée Hulu** : Netflix audience plus skip-resistant mais aussi plus cynical sur creative obvious-ad. Adapter le creative.
7. **Ignorer la fragmentation 2026** : il n'y a pas un seul "retail media network". Chaque retailer = setup, data, mesure différents. Sous-estimer = waste.

## Mesure / KPIs

**CTV/OTT** :
- CPM : $25-65 selon premium niveau.
- VCR (Video Completion Rate) : >85% benchmark.
- Incremental conversion lift via MMM ou holdout test : minimum 5-15% lift.
- Brand lift survey si spend >$200K.

**TikTok Shop Ads** :
- ROAS : 2-4× target DTC. <1.5× = revoir creator strategy.
- CPM : $5-20.
- Conversion rate in-app : 1-3% (très variable par produit).

**Retail Media** :
- ROAS retailer-reported : 3-8× pour Sponsored Search, 2-5× Display.
- iROAS (incremental) : viser >70% des ventes attribuées comme incrementales (holdout test).
- Share of Voice on retailer.com search.

**Programmatic DOOH** :
- Impressions geo-fenced + reach.
- Mobile lift study : 5-20% lift sur exposed vs control.
- Pas de KPI click-based pertinent.
