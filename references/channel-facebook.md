# Facebook (organique)

> **Channel scope** : ce framework couvre Facebook **organique** uniquement — Pages, Groupes, FB Live, Marketplace. **Ne couvre PAS** : FB Ads (→ `promote-faris-meta-and-fundamentals` qui distille la doctrine paid Meta), ni Instagram (canal séparé), ni WhatsApp (→ `channel-emerging-platforms-2026`).
> **Why framework, not operator** : canal **organique en déclin** depuis 2018 (zero-reach algo Meta), pas d'operator récent distillable spécifiquement sur FB organic 2026. Mari Smith reste référence historique mais sa doctrine date pré-2020. Le canal n'a plus de figure dominante — il est en mode **maintenance** pour use cases spécifiques.
> **Used by** : `promote-channel-strategist` quand le mix inclut FB organic — typiquement audiences 45+, local services, niche communities, ou stratégie B2B group-driven.

## Quand consulter ce framework

**Fits** :
- ICP **45+** (FB reste la plateforme dominante de cette tranche d'âge en 2026).
- **Local services** (plombier, kiné, coach local) où FB Marketplace + Groupes locaux = lead gen direct.
- **Communautés niches déjà établies** sur FB Groupes (parents, certaines verticales B2B, hobby groups 50k+ members).
- **FB Live** pour use cases trust-building (consultants, coaches, formateurs avec audience existante).

**Anti-fits** :
- ICP **<35** (Gen Z, Millennials < 35 = quasi-absents de FB en 2026).
- B2B SaaS / tech / dev tools (LinkedIn + X + GitHub > FB par 100x).
- Lancement de produit nouveau sans communauté pré-existante (FB organic reach = ~1-3% sur Pages, tu prêches dans le vide).
- Marques cherchant de la **growth scalable** — FB organic n'est pas scalable en 2026, période.

## Tactiques 2026

### 1. Pourquoi FB organic est largement mort en 2026

À déclarer franchement avant tout :

- **Pages reach 2026** : 1-3% des followers voient un post Page (down from 16% en 2012).
- L'algo priorise content "from friends and family" et inflation des Reels (concurrence avec TikTok/IG).
- Post de Page sans budget paid sponsoring = quasi-invisible.
- **Conséquence stratégique** : ne JAMAIS recommander FB Pages comme primary organic channel en 2026, sauf use cases ci-dessous.

### 2. FB Groupes (le seul vrai vivier organique restant)

Les groupes ont **encore** du reach organique parce que l'algo prioritise la group activity dans le News Feed des members.

- **B2B** : groupes professionnels niche (HR France, Marketing Agency Owners, SaaS founders FR) — souvent 5-50k members très actifs.
- **Community founders** peuvent build leur own group : 200-2000 members engagés > 50k followers Page.
- **Pattern** : le group founder devient l'autorité de facto. Build trust longue durée via posts valeur + modération.

Tactiques :
- Pose une **règle "no self-promo" stricte** sauf threads dédiés (jeudi promo day classic).
- Founder-driven posts (vrai prénom-photo, pas brand) > Page-driven posts.
- Live Q&A monthly = reach amplification.

### 3. FB Live for trust use cases

Live garde une priorité algo Meta (Live > video > photo > text en reach pondéré).

- Use case : consultants/coaches/formateurs faisant des **office hours live** mensuels.
- 30-60 min, Q&A driven, founder face caméra.
- Replay reste sur la Page mais le **vrai value** est le live notification push aux followers (l'un des derniers reach push gratuits).

### 4. FB Marketplace lead gen (B2C local)

Marketplace 2026 reste sous-exploité par les marketers, plein potentiel pour :

- **Local services** (déménagement, jardinage, garde enfants, coaching local).
- **B2C product avec stock physique** (vintage, handmade, equipment).
- Listing avec photo qualité + price + DM disponible = leads directs sans coût pub.
- Cross-post auto vers FB local groups ("buy/sell [city]") amplifie reach.

### 5. Mari Smith methods (light, héritage)

Mari Smith (référence FB historique) garde quelques principes valides en 2026 :

- **3-2-1 rule** : 3 contenus valeur / 2 engageants (questions, polls) / 1 promo.
- **Prime time** : 1pm-3pm + 7pm-9pm dans le timezone audience (encore valide pour Groupes).
- **Conversation > broadcast** : posts qui posent une vraie question génèrent comments → algo boost.
- **Reciprocity** : commenter les posts de tes members renforce la présence dans leur feed.

Ces principes sont 2014-vintage mais robustes là où FB organique vit encore (groupes).

### 6. Older audiences (45+) playbook

Si ton ICP est 45+, FB reste plateforme #1 :

- **Pages** marchent un peu mieux ici (l'algo News Feed des 45+ est moins saturé Reels).
- Posts longform (200-400 mots) avec photo personnelle authentique > short snappy.
- Photo > vidéo > link pour cette audience (counter-intuitif vs algo 2026 général).
- Mentions famille / community / valeurs = engagement amplification.

## Cadence + format recommandé

- **Volume hebdomadaire** : 2-3 posts/semaine en Page (au-delà : déclin engagement par post). Dans Groupes : daily activity (commentaires + 1 post/semaine).
- **Formats prioritaires** : (a) FB Live ; (b) posts longform dans Groupes ; (c) Marketplace listings ; (d) photos personnelles authentiques (45+ audience).
- **Anti-formats** : link posts (algo penalize external links), reposts cross-Meta non-adapté (un post IG balanced direct sur FB sans adapt = mort).

## Anti-patterns 2026

1. **"FB pour tout"** : recommander FB comme canal primary pour B2B SaaS, dev tools, tech audience moderne = malpractice strategist 2026.
2. **Pages sans paid budget** : tu postes sur ta Page Brand sans sponsoring = 1-3% reach = écrire dans le vide. Soit tu paies, soit tu skip Pages.
3. **Cross-post automatique IG → FB** : Meta-encouragé mais le format IG (Reel vertical, hashtags IG) performe mal sur FB. Adapt ou skip.
4. **Spam dans Groupes** : poster ta promo dans 50 groupes = ban en chaîne (groupes ont des admins humains qui détectent en 24h).
5. **FB Live sans promotion préalable** : live sans audience = 3 viewers. Promotionne 1 semaine en amont via posts groupes + email.
6. **Investissement temps disproportionné** : passer 5h/semaine sur FB organic pour 50 reach total = misallocation. Reallocate vers LinkedIn ou paid.
7. **Mari Smith methods 2014 sans adaptation** : appliquer 3-2-1 rule sur Pages 2026 sans paid sponsoring = échec garanti. Ces methods marchent UNIQUEMENT dans Groupes ou avec paid amplification.

## Mesure / KPIs

- **Group engagement rate** : (comments + reactions) / members * 100. >5% = group sain. <1% = mourant.
- **Live concurrent viewers** + replay views 7j (proxy trust audience).
- **Marketplace lead volume** : DMs / listing.
- **Page reach %** : si <2%, signal que Pages organique ne marche plus pour ce business.
- Benchmarks 2026 : Page organic reach = 1-3% ; Group post reach = 30-60% ; Live notification push = 10-20% des followers notifiés.
