# Pinterest (organique)

> **Channel scope** : ce framework couvre Pinterest **organique** — Pins, Idea Pins (ex-Story Pins), Boards, visual SEO, link-back-to-blog mechanics. **Ne couvre PAS** : Pinterest Ads (→ paid framework dédié), ni l'export multi-canal automatique depuis IG (option Meta-side).
> **Why framework, not operator** : Pinterest **fonctionne sur niches très spécifiques** (DIY, mariage, cooking, mode, home decor) et n'a pas de figure dominante 2026. Le game est *visual SEO + niche dominance*, qui se transmet par best practices itératives plutôt que par doctrine d'opérateur.
> **Used by** : `promote-channel-strategist` quand le mix produit s'aligne avec verticales Pinterest-friendly (B2C lifestyle, e-commerce niche, content-driven blog avec affiliation).

## Quand consulter ce framework

**Fits** :
- ICP **majoritairement féminin 25-55** (62% audience Pinterest 2026, et le power-user demographic).
- Verticale **DIY, wedding, cooking, home decor, mode/style, beauty, parenting, gardening, fitness/wellness, travel inspiration**.
- Business model qui bénéficie du **traffic blog evergreen** (affiliation, e-commerce avec SEO sous-jacent, content monetization).
- Capacité à produire du **visuel calibré Pinterest** (vertical 2:3, branded, lisible mobile).

**Anti-fits** :
- ICP **B2B / dev / tech** : Pinterest n'a quasiment pas cette audience.
- Produits **abstraits / services** sans visuel naturel (consulting, SaaS infra, legal services).
- Business cherchant du **traffic immédiat** : Pinterest est un **canal long-terme** (un Pin met 3-6 mois à indexer/croître).
- Marques avec budget visuel zero : Pinterest = marché visuel, des pins moches = invisibles.

## Tactiques 2026

### 1. Verticales qui marchent (ranking 2026)

Par ordre de puissance organique 2026 :

1. **Wedding / event planning** : Pinterest reste plateforme #1 inspiration mariage, bonne CTR vers blogs/sites.
2. **Home decor + interior design** : audience massive, intent achat élevé, parfait pour e-commerce déco.
3. **DIY / crafts** : evergreen massive, content-driven, monetization via blogs + affiliation.
4. **Cooking / recipes** : recettes visuelles génèrent trafic blog massif (food bloggers — pinterest = >70% de leur trafic souvent).
5. **Mode / style / beauty** : strong, mais concurrencé par TikTok/IG en 2026.
6. **Gardening / plant care** : niche en croissance.
7. **Fitness / wellness / nutrition** : moyen, dépend de la spécificité.
8. **Travel inspiration** : mood boards + city guides marchent bien.

Hors ces verticales, Pinterest est un canal secondaire au mieux.

### 2. Idea Pins (ex-Story Pins)

Format prioritisé 2026 par l'algo Pinterest :

- **Vertical 9:16**, multi-pages (3-7 pages typique), texte overlay lisible.
- Contrairement aux Stories IG/TikTok, Idea Pins sont **persistantes** (pas 24h).
- Format idéal : **mini-tutoriel step-by-step** (recette 5 étapes, DIY tutorial, before/after déco).
- Pas de lien sortant direct (Pinterest pousse l'engagement in-platform), mais bio + Pinterest profile drive vers site.

### 3. Visual SEO (le coeur du game Pinterest)

Pinterest = **moteur de recherche visuel**. Le SEO Pinterest = même logique que Google SEO :

- **Keyword-rich Pin titles** (100 chars max) : "DIY [keyword] tutorial easy [variation]".
- **Description 500 chars** : keywords naturellement intégrés, jamais stuffed. Inclure variations sémantiques.
- **Board names** : keywords-driven ("Boho Bedroom Decor Ideas" > "My Style"). Renomme tes boards.
- **Alt text** sur le pin (Pinterest accessibility-driven mais aussi keyword indexed).
- **Recherche keywords** : utilise Pinterest search bar autocomplete (donne intent réel users).

### 4. Link-back-to-blog mechanic

Pattern dominant 2026 (food, DIY, wedding bloggers vivent de ça) :

- Pin → click vers blog post → blog monétisé (display ads + affiliation + email capture).
- Crée 5-10 Pins **différents visuellement** par blog post (variation A/B sur design, hook visuel, title).
- Chaque Pin pointe vers le même URL.
- Pinterest aime la diversité visuelle pour un même URL (signal de "popular content").

### 5. Evergreen traffic potential

Différence majeure vs Twitter/IG/TikTok :

- Un Pin peut générer du traffic **3 mois → 5 ans** post-publication.
- Les Pins "viraux" continuent à indexer et grow (saved → re-distributed dans les feeds des saveurs).
- Calcul ROI : 1 Pin = 1 mini SEO asset, qui se compose dans le temps.
- → stratégie pertinente uniquement sur **horizon 12+ mois**. À 3 mois tu vois rien.

### 6. Niche dominance > broad audience

L'erreur classique : essayer de plaire à toute audience Pinterest. Le pattern qui marche :

- **Spécialisation extrême** ("boho bedroom décor for small apartments") > generalist ("home decor").
- Build 5-10 boards ultra-niche, 50-100 Pins par board, all on-brand.
- Devenir l'autorité visuelle de cette niche → algo Pinterest favorise les comptes "topic authority".

### 7. Pin design templates 2026

Patterns qui performent :

- **Vertical 1000x1500 (2:3 ratio)** = format standard. Anti-format : 1:1 square.
- **Texte overlay 30-50% surface** : hook + value claire en glance.
- **Couleurs branded** consistent à travers tous tes pins (Pinterest reconnaît visual identity).
- **Photos lifestyle** > stock photos (perform 2-3x mieux).

## Cadence + format recommandé

- **Volume hebdomadaire** : 5-15 Pins/semaine recommandé en 2026 (down from 25-50/semaine 2019, l'algo a recalibré contre le spam). Tailwind/Pinterest scheduler aide.
- **Formats prioritaires** : (a) Idea Pins step-by-step ; (b) Static Pins haute qualité avec link-back ; (c) Video Pins courtes (15-30s, recipe ou tutorial).
- **Anti-formats** : pins horizontaux, pins basse résolution, pins avec watermark visible (Pinterest pénalise les watermarks d'autres plateformes).

## Anti-patterns 2026

1. **Repost massif depuis IG** : importer ses posts IG carrés sans adapter au format Pinterest (vertical 2:3) = invisibilité.
2. **Pins sans texte overlay** : la photo seule fait pas le job. Pinterest est un moteur de recherche, le texte = signal.
3. **Hashtag stuffing** : Pinterest n'utilise plus les hashtags depuis 2022. C'est le keyword-rich description qui compte.
4. **Pins pointant vers homepage** : pin doit pointer vers une **landing page spécifique** ou blog post précis. Homepage = bounce massif.
5. **Spray-and-pray pinning** : 100 pins random/jour = algo penalize comme spam. 5-15 pins ciblés > 100 pins random.
6. **Boards génériques** ("Inspiration", "Ideas", "Style"). Boards doivent être keyword-driven et niche.
7. **Abandonner à 3 mois** : Pinterest demande 6-12 mois minimum pour voir les résultats. Beaucoup arrêtent à 3 mois et concluent "ça marche pas".

## Mesure / KPIs

- **Outbound clicks** (clic Pin → site) : KPI principal Pinterest. Pinterest Analytics donne ce metric directement.
- **Saves** (pins sauvegardés par d'autres users) : signal algo de qualité, prédicteur de growth.
- **Impressions** (combien de fois le Pin est vu) : volume top-funnel.
- **Total monthly viewers** (account-level metric Pinterest) : 100k+ = small, 1M+ = grosse autorité niche, 10M+ = top niche player.
- **Referral traffic site** (Pinterest → site) via GA4 : pour bloggers food/DIY, peut représenter 50-80% du traffic total.
- Benchmarks 2026 : 1 Pin "qui marche" = 5-50k impressions, 50-500 saves, 50-500 outbound clicks sur sa durée de vie totale.
